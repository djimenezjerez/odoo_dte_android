### Paso #1: Configurar dirección de servidor ODOO.
-   Ingresar la dirección del servidor ODOO.
-   Haga clic en el boton guardar.
- __NOTA:__ si la conexión es exitosa, las próximas ocaciones que ingrese a la aplicación ya NO realizará este paso.

![Dirección url](/Docs/assets/url.png)

### Reintentar conexión:
-   Si después de realizar el [Paso #1: Configurar dirección de servidor ODOO](#paso-1-configurar-dirección-de-servidor-odoo), no se logró conectar al servidor odoo, se le habilitará las siguientes opciones: __Reintentar conexión, configuración__.

-  __Reintertar conexión__ :  si hace clic sobre esta opción, se intentará conectar nuevamente al servidor odoo.
-   __Configuración__ : Si hace clic sobre esta opción se le habilitará  el formulario indicado en el [Paso #1: Configurar dirección de servidor ODOO.](#paso-1-configurar-dirección-de-servidor-odoo)
